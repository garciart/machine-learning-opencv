Windows:
-- Must use Python 3
-- Must pip install as administrator.
-- Shapely depends on GEOS. The following steps will install the required geos libraries:
1. Install Wheel (if not installed): pip install wheel
2. Uninstall Shapely (if installed): pip uninstall Shapely
3. Download the appropriate wheel (based on Python version) into the Park directory from https://www.lfd.uci.edu/~gohlke/pythonlibs/#shapely
4. Install the .whl file: pip install Shapely-1.6.4.post2-cp36-cp36m-win_amd64.whl
